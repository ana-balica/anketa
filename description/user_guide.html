User Guide

Use Case 1 - User authentication

In order to authenticate yourself you will need to fill out the email and password data in the top right corner form available on every page. After that click Sign in and you are in.

Use Case 2 - Create a new survey

In order to create a new survey the user can click on Create a poll on the main page of website or in the top left corner on any page. User is redirected to the page which contains the form Create a poll. User should enter the title, description (optional) and questions. From a drop-down list user chooses a question type (Text question, Single choise question, Multiple choise question). If user selects Text question, then a single text input appear. If user selects Single choise question, then a text input for question appears and a plus sign to add answer choices. Similar scenario is available for Multiple choice answer, the only difference is that the answer choices are going to be checkboxes. After the user is done creating the poll, click on Create poll button at the bottom.

Use Case 3 - Edit a survey

Go to the list of polls and locate the poll that needs to be edited. Near the poll in the top right corner there is a small button called Edit. Click on Edit and make the appropriate changes. When done, click Save at the bottom of the edit page.

Use Case 4 - Display results of a survey

Go to the list of polls and locate the poll the results of which you want to view. Click on the button View results. User will be redirected to a page which contains the results.

Use Case 5 - Delete a survey

Go to the list of polls and locate the poll that needs to be deleted. Near the poll in the top right corner there is a small button called Delete. Click on Delete. A popup will appear asking if the user is sure he wants to delete the poll, if yes - the poll the deleted.

Use Case 6 - Take a poll

Go to the list of polls. Choose a poll that you want to take. Fill out the necessary answers and click Submit.
